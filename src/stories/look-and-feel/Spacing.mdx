# Spacing

The spacing scale, drawn to size:

<div style={{ display: 'flex', flexDirection: 'column', gap: '.5rem', marginBlock: '1.5rem' }}>
	{[
		['xs', '0.25rem'], ['s', '0.5rem'], ['m', '0.75rem'], ['l', '1rem'], ['xl', '1.25rem'],
		['2xl', '1.5rem'], ['3xl', '1.75rem'], ['4xl', '2rem'], ['5xl', '2.5rem'],
	].map(([name, value]) => (
		<div key={name} style={{ display: 'flex', alignItems: 'center', gap: '1rem' }}>
			<code style={{ inlineSize: '3.5rem' }}>{name}</code>
			<div style={{ blockSize: '1rem', inlineSize: value, background: '#2f6df6', borderRadius: '4px' }} />
			<span style={{ color: '#6b7280' }}>{value}</span>
		</div>
	))}
</div>

Every gap, padding and margin on the site comes from this one scale — never a loose pixel value. The
steps are named `xs` through `5xl`, and you can also count in numeric steps `0`–`8`, where each step is
`0.25rem`. So `spacing(2)` equals `spacing(s)` equals `0.5rem`.

## How to use it

You reach for the scale through the `spacing()` function in SCSS:

```scss
.card {
	padding: spacing(xl);            // 1.25rem
	gap: spacing(m);                 // 0.75rem
	margin-block-end: spacing(4xl);  // 2rem
}
```

> **Why a single scale?** When every distance is one of nine steps, the whole site shares one rhythm
> and nothing looks slightly off. Change the scale in one place and every margin, padding and gap moves
> with it — there are no stray `13px` values to hunt down.

Where it lives: `src/styles/_initial.scss` (the `$spacing` map). The named steps are a map; the numeric aliases are a
formula (`step × 0.25rem`), so they're never duplicated.

## Breakpoints

Spacing and type respond at a shared set of breakpoints (used through the `bp()` mixin, all
mobile-first min-widths):

| Name | Min width |
|---|---|
| `s` | 24rem |
| `m` | 48rem |
| `l` | 64rem |
| `xl` | 80rem |
| `2xl` | 90rem |
| `3xl` | 120rem |

```scss
.grid {
	gap: spacing(m);

	@include bp(l) {
		gap: spacing(xl); // wider gap from the large breakpoint up
	}
}
```

Where it lives: `src/styles/_initial.scss` (the `$breakpoints` map).
